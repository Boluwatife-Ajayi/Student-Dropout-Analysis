# Student-Dropout-Analysis
Executive Summary
Student dropout is a persistent challenge in higher education, affecting students' academic success, institutional performance, and long-term career opportunities. The solution demonstrates how educational data can support
evidence-based decision-making by enabling earlier interventions and
more efficient allocation of institutional support resources.

# Objectives
* Identify factors contributing to student dropout
* Explore relationships between academic, financial and social variables
* Build an interactive executive dashboard
* Recommend strategies for improving student retention

# Tools Used
* Microsoft Excel
* SQL
* Python
* Jupyter Notebook
* Power BI Desktop
